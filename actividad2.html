2.1- ¿qué es una url absoluta?

Una url absoluta es una cadena de texto que describe completamente la ubicación de un archivo en la web

2.2- ¿qué es una url relativa?

Las URL relativas describen la ruta a un archivo relativo al actual documento, estas se pueden usar cuando se vincula a otro documento en su propio sitio
